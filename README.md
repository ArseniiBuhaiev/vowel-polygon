# Генератор формантних полігонів (Vowel Polygons Generator)

Веб-застосунок для візуалізації вокальних просторів та побудови формантних полігонів голосних звуків. Інструмент розроблений для лінгвістів, фонетистів та студентів філологічних факультетів.

## 🚀 Функціонал
- **Динамічне додавання мовців:** Вводьте значення формант F1 та F2 для кожного голосного звука.
- **Еталонне порівняння:** Автоматичне накладання даних на еталонну сітку (за методикою Іщенка).
- **Професійна візуалізація:** Побудова графіків за допомогою `matplotlib` з урахуванням фонетичних стандартів (інвертування осей F1/F2 для ілюстрації залежності позиції язика зі значеннями формант).
- **Порівняльний аналіз:** Можливість додавати кількох мовців на один графік для порівняння артикуляційних баз.
- **Список даних:** Перегляд та керування введеними значеннями у зручному інтерфейсі.

## 📋 Плани
Рушійною ідеєю проєкту є автоматизація формантного аналізу голосів мовців, тому в пріоритеті наступні покращення:
- **Автоматизація вимірювання формант:** Відсутність у потребі вводити дані вручну, натомість програма робитиме це на основі аудіозапису
- **Автоматичний аналіз даних:** Розрахунок ймовірності, що показники належать тому самому мовцеві.
